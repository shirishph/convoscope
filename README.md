# convoscope
Visual aid for conversation designers


## Installation 
- ##### Server
```
$ cd {CONVOSCOPE_DIR}/back/ 
$ python3 -m venv venv
$ source ./venv/bin/activate
$ pip install -r requirements.txt
```

- ##### User-interface
```
$ cd {CONVOSCOPE_DIR}/front/
$ npm install
```


## Usage 
- ##### Server
```
$ cd {CONVOSCOPE_DIR}/back/
$ source ./venv/bin/activate
$ python3 app.py
```

- ##### User-interface
```
$ cd {CONVOSCOPE_DIR}/front/
$ npm start
```


## Start using
Open `http://localhost:3000/`
